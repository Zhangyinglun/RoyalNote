按钮紧邻起止时间，但保持次级层级。无论应用打开多久，轻点一次即可把开始和结束时间同时更新为当前时刻，再按需微调。

推荐样式

  • 文案用“设为现在”，含义直接，不需要学习。
  • 铜色描边、小钟表图标；主操作“入录”仍是唯一实心按钮。
  • 放在“起止时间”标题右侧，避免挤压两个日期与时间入口。
  • 点击后短暂显示“已更新”，提供明确反馈。
  • 视觉保持轻巧，实际 Android 触控区域仍至少为 48dp。
